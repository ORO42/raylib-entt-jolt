# Prerequisites

Before you begin, ensure you have the following installed:

- A C++ compiler
- CMake

## Instal/Build/Run

1. Clone the repo.
2. Navigate to project folder

```
mkdir build
cd build
cmake ..
make
./raylib-entt-jolt
```

### VSCode

Ensure correct C++ standard is set in `.vscode/c_cpp_properties.json` (e.g. `"cppStandard": "c++17",`)
